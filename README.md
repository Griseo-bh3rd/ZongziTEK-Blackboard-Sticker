## 小黑板
- 目前支持横细竖粗，但未完成根据速度和墨迹尾部的笔锋识别
- 墨迹自动保存，打开时自动读取
- 保存的墨迹文件支持使用WXRIW的Ink Canvas画板软件打开
## 启动台
- 打开绝对路径里面的希沃视频展台、希沃白板5、资源管理器
## 课程表
- 判断星期几，自动切换每天课程
- 带有课程表编辑器
- 以json方式存储
